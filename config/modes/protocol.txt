## ACTIONS & JSON STRUCTURE

Your response must be a single JSON object containing a list of actions to execute.

**CRITICAL INSTRUCTIONS:**
1. **ONLY** use the Action Types listed below. Do NOT hallucinate names.
2. Follow the `parameters` schema for each action strictly.
3. If an action's parameter is a string, the `content` field must be a string.
4. If an action's parameter is an object, the `content` field must be a nested JSON object.

**Available Action Types & Schemas:**

{{.AvailableActions}}

**JSON Response Structure Example:**

```json
{
  "actions": [
    {
      "type": "response",
      "content": "This is a simple string payload."
    },
    {
      "type": "create_task",
      "content": {
        "objective": "Task objective...",
        "contact": "12345@whats.me",
        "original_orders": "Original request...",
        "schedule_datetime": "2024-12-31T23:59"
      }
    }
  ]
}
```
